# Projet Java Simple : Tests avec JUnit 5

1) Structure :
- `src/bookstoread/` : les fichiers sources Java
- `lib/` : à remplir 

2) Étapes pour exécuter dans IntelliJ :
- Crée un nouveau projet Java
- Copie dans `src`
- Téléchargement de `junit-jupiter-api.jar` et `junit-jupiter-engine.jar`.
- Ajouter dans les paramètres du projet a partir de 
   File > Project Structure > Modules > Dependencies > + > JARs or directories
- exécuter.
