# Projet Java Simple : Tests avec JUnit 5

## Structure :
- `src/bookstoread/` : les fichiers sources Java
- `lib/` : à remplir avec les fichiers JAR de JUnit 5 (non inclus ici)

## Étapes pour exécuter dans IntelliJ :
1. Crée un nouveau projet Java simple.
2. Copie les fichiers dans `src`.
3. Télécharge `junit-jupiter-api.jar` et `junit-jupiter-engine.jar`.
4. Ajoute-les dans les paramètres du projet via :
   File > Project Structure > Modules > Dependencies > + > JARs or directories
5. Clique sur le triangle vert à côté du test pour l'exécuter.
